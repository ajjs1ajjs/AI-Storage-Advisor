# AI Storage Advisor 🚀

**AI Storage Advisor** — це надлегкий, безпечний та високопродуктивний настільний додаток для аналізу стану дисків, прогнозування вичерпання вільного місця та автоматичного очищення сміття за допомогою локального або хмарного штучного інтелекту.

Проект успішно мігровано з Python (PySide6) на **Wails (Go + HTML/JS/CSS)** для забезпечення блискавичної роботи, мінімального використання оперативної пам'яті (до 50-70 МБ) та нульової залежності від CGO (чиста крос-компіляція для Windows).

---

## 🛠️ Архітектура та Технології

Додаток побудований за сучасною гібридною архітектурою:
* **Frontend**: HTML5, Vanilla JS (ES6+), CSS3 (вбудований темний/світлий дизайн з плавними мікро-анімаціями, Outfit шрифти та кастомні SVG-графіки для показників SRE Health). Створено за допомогою Vite.
* **Backend**: Go (Golang) — керує всіма системними викликами, паралельним скануванням файлових систем та безпекою.
* **База даних**: SQLite (за допомогою pure Go драйвера `modernc.org/sqlite` — не потребує CGO). Зберігає історію сканувань, профілі та налаштування.
* **Криптографія**: Вбудований менеджер сейфа (Vault) на базі шифрування AES-256-GCM з деривацією ключів Argon2id для надійного збереження API-ключів та SSH-паролей у SQLite.

## ✨ Основні можливості

1. **Глибоке сканування дисків (Local / UNC / SSH)**:
   * Сканування локальних накопичувачів та мережевих папок (UNC-шляхів).
   * **Віддалене сканування Linux (SSH)**: аналіз використання дисків на віддалених серверах за допомогою команд `df`, `find` та `md5sum`.
2. **SRE Health Score (Показник здоров'я системи)**:
   * Оцінка ефективності використання сховища від 0 до 100.
   * Виявлення сміття в Docker (розміри шарів контейнерів, невикористовувані томи).
   * Перевірка системних папок Windows (системні дампи, файли глибокого сну тощо).
3. **Smart Duplicate Checker**:
   * Надшвидкий алгоритм пошуку дублікатів файлів за префіксним хешуванням та повним MD5-порівнянням для запобігання колізіям.
4. **Прогнозування вичерпання диска (OLS Forecast)**:
   * Хронологічний аналіз розміру диска за допомогою лінійної регресії (метод найменших квадратів) для прогнозування дати вичерпання вільного місця.
5. **Інтеграція з ШІ (AI Providers)**:
   * Підтримка локальних ШІ (Ollama, LM Studio) та хмарних сервісів (OpenAI, Gemini, Anthropic, DeepSeek, Custom API).
   * Генерація структурованих рекомендацій з очищення українською мовою.
   * Автоматичне формування безпечних посилань `delete://` для швидкого видалення сміття безпосередньо з порад ШІ.
6. **Безпечне видалення (Safe Delete)**:
   * Інтеграція з кошиком Windows за допомогою викликів функції `SHFileOperationW` (бібліотека `shell32.dll`).

## 🚀 Як зібрати додаток

### Вимоги:
* Встановлена Go (Golang) версії 1.20+.
* Node.js (версії 16+).
* Встановлений Wails CLI:
  ```powershell
  go install github.com/wailsapp/wails/v2/cmd/wails@latest
  ```

### Збірка:
Перейдіть у корінь проекту та запустіть команду:
```powershell
wails build
```
Готовий виконуваний файл Windows буде створено за шляхом:
`D:\PROJECT\AI-Storage-Advisor\build\bin\aisadvisor.exe`

## 🔒 Ліцензія та Безпека
Усі збережені паролі до SSH та API-ключі шифруються за допомогою сесійного ключа AES-256-GCM. Програма не передає жодних персональних даних третім сторонам, окрім вибраного вами ШІ-провайдера.
